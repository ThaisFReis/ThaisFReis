## Current Focus

- **Karn Protocol**: contribution-weighted governance on Stellar/Soroban using non-transferable reputation (SBT-like badges), **Mana** (voting influence), and **time decay** to prioritize active contributors. Protocol modules: Valocracy, Governor, Treasury.
- **Karn Ecosystem dApp**: a reference implementation with multi-wallet onboarding, identity gating, and governance-gated treasury flows.
